Diese Website wird über GitHub Pages bereitgestellt, einem Dienst der GitHub, Inc., 88 Colin P. Kelly Jr. St, San Francisco, CA 94107, USA.

Beim Abruf der Website erfasst GitHub automatisch sogenannte Server-Logdaten. Diese können enthalten:

4. Eingebettete YouTube-Videos

Auf dieser Website werden YouTube-Videos eingebunden. Betreiber ist Google Ireland Limited, Gordon House, Barrow Street, Dublin 4, Irland.

Die Videos werden zunächst nur als Vorschaubild dargestellt. Das Vorschaubild wird von unserem eigenen Server ausgeliefert – es findet dabei keine Verbindung zu YouTube-Servern statt. Erst wenn Sie aktiv auf das Vorschaubild klicken und damit Ihre Einwilligung erteilen, wird das Video über youtube-nocookie.com geladen. Erst zu diesem Zeitpunkt kann YouTube Daten verarbeiten und Cookies setzen.

Weitere Informationen: Google Datenschutzerklärung
